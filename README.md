## Readme file styling

**Heading**
```
example: # learn-git-readme
         ## learn-git-readme
         _ _ _ _ _
         ###### learn-git-readme
```
> output:
> 
# learn-git-readme
## learn-git-readme
###### learn-git-readme

**Bold text** 
```
command: ctrl + b
example: **Bold text**
```
> output: **this is bold text**\

**Italic text** 
```
command: ctrl + i
example: *Bold text*
```
> output: *this is bold text*\

**Strikethrough Text**
```
example: ~~Bold text~~
```
> output: ~~this is bold text~~\
