## Readme file styling

**Bold text** 
```
command: ctrl + b
example: **Bold text**
```
> output: **this is bold text**\

**Italic text** 
```
command: ctrl + i
example: *Bold text*
```
> output: *this is bold text*\

**Strikethrough Text**
```
example: ~~Bold text~~
```
> output: ~~this is bold text~~\
